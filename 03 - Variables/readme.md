# Variables

## Var

Implicit:

```csharp
var x = 1;
```

Explicit:

```csharp
int x = 1;
```

Can be changed:

```csharp
int x = 1;
x = 2;
```

## Const

Always explicit

```csharp
const int x = 1;
```

Cannot be changed:

```csharp
const int x = 1;
x = 2; // Error: The left-hand side of an assignment must be a variable, property or indexer
```
